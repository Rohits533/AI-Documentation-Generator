# 📚 AI Documentation Generator

An AI-powered tool that auto-generates READMEs, docstrings, and API documentation from your code using AST parsing + Groq AI.

## 🚀 Features

- **AST Parsing** – Extracts functions, classes, and docstrings from Python code
- **README Generator** – Creates a professional README from your project structure
- **Docstring Generator** – Writes Google-style docstrings for every function
- **AI Explanations** – Groq explains what your code does in plain English
- **Streamlit UI** – Clean, dark-themed interface
- **One-Click Download** – Download generated docs as a `.md` or `.txt` file

## 🛠️ Tech Stack

- Python
- AST (Abstract Syntax Tree)
- Groq API
- Streamlit
- `astunparse`
